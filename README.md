# Maven

### Q - What is POM 
==> Project Oriented Model

### Minimum Requirement for POM 

- **Project** - root
- **Model Version** - 4.0.0
- **group id** - com
- **artifact** - vdoxx
- **version** - 2.1
- **Packaging** - war   

### Sample 

```
<project> 
    <model version>**4.0.0**</model version>
    <group id >**com**</group id>
    <artifact id>**vdoxx**</artifact>
    <version>**2.1**</version>
</project>
```
